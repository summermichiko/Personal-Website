Butterbean

Go to project
Mongo DB, Express, Angular, Node.js
Butterbean is an online directory that features all of Hawaii’s best activities and services. Parents are able to use the interface to quickly search and filter for exactly what they’re looking for. Each profile is robust and picture heavy. This project is still a work in progress. The only complete profile is of Punahou School.

1. For consistency, only clickable items are highlighted in red.

2. Because so much information is being displayed visually, the app adheres to a strict hierarchy. Each repeated sub-section has uniform styling.

3. Images are orgaizned to maximize image size while not overwhelming the screen.

4. Related profiles are displayed at the end of the page to keep users engaged on the site.

5. This app is fully responsive, designed to look great on all devices.
