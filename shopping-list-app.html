Shopping List App

Go to project
Javascript, jQuery, HTML5, CSS3
Shopping List App is a front-end app that allows users to quickly compile a grocery shopping list.

1. The flat vector, hero image was created from scratch.

2. A user can add a new grocery item by either clicking on the 'Add' button or pressing 'enter'. The newly added item softly animates in.

3. Users are able to check and uncheck grocery items without having to remove them from the list. Part of the satisfaction is seeing what's been accomplished.

4. The Shopping List App is fully responsive. In the mobile optimized version, the input bar is more aggresively highlighted to clearly point the user to the most important action.
