Hot or Cold Game

Go to project
Javascript, jQuery, HTML5, CSS3
Hot or Cold Game is a web game where players have 5 tries to guess a random number between 1 and 100, prechosen by the computer. After each guess, the game alerts the user if they are hot (close to the number) or cold (far from the number).

1. Users can make a guess by either clicking on the 'Guess' button or pressing 'enter'.

2. After each guess, the app gives feedback on how "hot" or "cold" a user's guess is to keep them engaged.

3. A user's past guesses are listed to show how much longer they have in the game and to keep them engaged.

4. At the end of the game, a "play again" button replaces the input bar. This minimizes the number of actionable options a user has at any point in the game.

5. The Hot or Cold Game is fully responsive and designed to work for all devices.
